What is JSX, and why is it used?
JavaScript XML, XML within JavaScript. It is a syntax extension for JavaScript, not a new language.

What is the difference between State and Props?
State is data that is managed within a component, while props are data that is passed from a parent component to a child component.

What is the useState hook, and how does it work?
useState is a hook that allows you to add state to a functional component. It returns an array containing the current state value and a function to update it.

How can you share state between components in React?
You can share state between components in React by lifting the state up to a parent component and passing it down to child components as props.

How is event handling done in React?
Event handling in React is done by using event handlers, which are functions that are called in response to user interactions such as clicks, key presses, and form submissions.
